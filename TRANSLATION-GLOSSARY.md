# Translation glossary (无限TRPG → English)

Canonical zh→en terminology for translating the rules. Every translated page
must use these renderings consistently. **The primary source of truth is the
old site's existing English translations**
(`GuonuoTW.github.io/ss/root/Rules/...`) — where the old site established a
term, that term wins. Terms the old site never translated fall back to
established WoD/D&D vocabulary.

Conventions:
- 【关键字】 item inscriptions render as bold brackets: **[Inscription]**.
- Dice notation stays as-is: DP (dice pool), DC, d10, XDP, AS.
- Damage/HP grades: H (Healthy), B (Blunt), L (Lethal), A (Atrophy/Aggravated);
  notation HP(B), DMG(L), 22L etc.
- "ST" (Storyteller = GM) and "PC" stay as-is.
