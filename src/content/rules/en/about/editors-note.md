# Editor's Note

This version of the rules is my revision and reorganization of version 2.361 (beta) by Lumia, which itself was built on version 2.35 — the final version updated by the original author, Xingguang Juemie.

This version is mostly patchwork: some rules and some resources have been revised — humble work, done only in the hope that players enjoy themselves. Matters of the rulebook should be settled within the rulebook: if you find any problem with it, please report it to the maintainer promptly so that I can keep improving this book. Once again, thank you, everyone, for your support and encouragement!

Version 2.44 update: most of the 2nd edition's poorly standardized or badly balanced resources have now been fixed. This may not be the most exciting derivative version, but this rulebook will keep updating for the players who need it, with balance and reasonableness as its guiding principles — avoiding numerical power creep as much as possible while encouraging players toward more functional, rather than purely numerical, abilities. That is roughly the style.
