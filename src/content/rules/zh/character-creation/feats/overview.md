# 专长概述

**专长概述：**

专长是人物拥有的一些特殊技巧，它不能单纯用技能和属性来概括，但又与主神处兑换的能力不同，它经过人物自身的磨练和追求来获得。

要学习一个专长需要：

为1级专长投入3点XP，

为2级专长投入6点XP，

为3级专长投入8点XP，

为4级专长投入12点XP，

为5级专长投入20点XP，

若一个专长拥有多个等级，你必须为每一等级单独学习，并且必须按顺序学习。

**因此，一个具有1-5点专长点数的分为5个等级的专长，一共需要3+6+8+12+20=49XP来完全掌握。**

若一个专长直接以高等级开始，则你直接花费对应等级的XP即可获得。

因此，一个具有2~3点专长点数的分为2个等级的专长，一共需要6+8=14XP来掌握。

对于基本前提和属性挂钩的专长，其等级不得超过该属性数值，例如耐力3的人不能有死硬5。若同时有两个关键属性，则以较低者作为上限。

对于战斗专长，你必须根据你的背景来选择战斗专长。即，如果你在特殊身份中描述自己是击剑俱乐部的会员，那么你没有理由选择“狙击大师”专长。

专长中未经描述的加值与减值均为专长加值与专长减值。
